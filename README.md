Kişisel web sitem...

## Rakefile

```bash
$ rake -T

rake build                                  # build pages
rake deploy[bump]                           # deploy to gh-pages with bump
rake new:page[title,language]               # new page
rake new:post[title,language,publish_date]  # new blog post
rake now                                    # print now
rake run_server                             # run server
```
